## 2. Beasiswa & Jurusan Finder

### 2.1 Schema (Prisma)

```
Major(id, slug, name, category, summary, tagR/I/A/S/E/C, isActive)
Scholarship(id, slug, name, provider, coverage, level, country,
            deadlineAt, applicationUrl, requirements, isActive)
ScholarshipMajor(scholarshipId, majorId) -- M:N
CareerProfile(studentId, scoreR/I/A/S/E/C, hollandCode, computedAt)
ScholarshipGuide + GuideArticle(slug, title, body, tag)
```

### 2.2 Algoritma matching

Quiz 18 pernyataan → 3 per dimensi RIASEC (R/I/A/S/E/C), skala 1-5,
normalisasi ke 0-100. Top 3 dimensi = Holland Code (mis. "ISC"). Matching
jurusan via cosine similarity vektor 6-dim user vs tag jurusan, sorted
descending, top 5 ditampilkan dengan persentase match.

### 2.3 Konten seed

- **28 beasiswa** seed di `scripts/seed-career.ts` + 11 hasil research Claude
  via prompt template (`scripts/ingest-research.ts`)
- **Jurusan** lengkap dengan tag RIASEC per major (kedokteran, teknik
  informatika, psikologi, manajemen, dll)
- Cakupan jenjang: S1 (KIP-K, Tanoto, GKS UG), S2 (LPDP, Chevening, AAS,
  Fulbright, MEXT, DAAD, Erasmus, Beasiswa Unggulan), S3 (LPDP, Fulbright,
  DAAD), negara: Indonesia, UK, Australia, AS, Jepang, Jerman, Korea, Eropa

## 7. Periodic Scraping (Apify + Brave + Claude)

### 7.1 Arsitektur

```
ScrapeSource (admin config) ──► trigger ──► runner
                                   │
                                   ├── Brave Search API   (URL discovery)
                                   ├── Apify actor        (HTML scrape)
                                   └── Anthropic Sonnet 4.6 (LLM extract → Zod schema)
                                          │
                                          └── upsert by slug → Scholarship/JobPathway/etc.
```

Prompt caching aktif (1-hour TTL) untuk schema definition dan instruksi
ekstraksi → cache hit di run kedua memotong biaya 90%.

### 7.2 Schema

```
ScrapeSource(id, name, kind, sourceUrl, targetSchema, claudeModel,
             isActive, lastRunAt, cronSchedule)
ScrapeRun(id, sourceId, startedAt, finishedAt, status, itemsExtracted,
          itemsUpserted, errorMessage, costEstimateUsd)
```

`kind` enum: `scholarship | job_pathway | guide_article | job_article`.

### 7.3 Admin Manager UI

`/admin/scrape` (page server) + `ScrapeManagerClient` (CRUD inline +
trigger button + status badge). Run history visible per source dengan
estimasi cost USD.

### 7.4 Cron

`/api/cron/scrape` (GET, cron-secret protected) — invoke semua source dengan
`cronSchedule != null` yang due. Manual trigger via admin UI tetap tersedia.

### 7.5 Research prompt template

`docs/RESEARCH_PROMPT_BEASISWA.md` — instruksi siap-copy untuk dikirim ke
claude.ai user-facing, output JSON terstruktur sesuai schema. Hasil
research masuk via `scripts/ingest-research.ts` (idempotent upsert).

## 9. IELTS Mock Test MVP

### 9.1 Schema

```
MockTest(id, slug, title, type, difficulty, description, isActive)
MockTestSection(id, mockTestId, skillType, sectionOrder, title,
                passageText, questionsJson, audioUrl)
MockTestAttempt(id, studentId, mockTestId, skillType, startedAt,
                finishedAt, rawScore, maxScore, bandScore, durationSec,
                answersJson)
```

`questionsJson` array — type `mcq | fill_blank | true_false_ng`.
`answersJson` simpan jawaban user + breakdown per section (untuk review).

### 9.2 Konten

1 test seed: **IELTS Mock Test Academic #1**
- Reading: 3 passage original (Coffee History, Vertical Farming,
  Bilingualism), 40 soal
- Listening: 4 section (Apartment Rental, Library Tour, Group Project,
  Urban Heat Islands), 40 soal

### 9.3 Band conversion (`src/lib/ielts-band.ts`)

Tabel resmi IELTS — Reading Academic vs Listening punya curve berbeda:
- Listening: 39-40 → 9.0, 37-38 → 8.5, ..., 11-12 → 4.0
- Reading: 39-40 → 9.0, 37-38 → 8.5, ..., 8-9 → 3.5

`gradeAnswers()` type-aware: MCQ exact match, fill_blank multi-valid via
pipe separator (`apartment|flat|unit`).

### 9.4 Test Runner

`/karir/ielts/[slug]/[skill]` — client component dengan:
- Timer countdown (60min reading, 30min listening), auto-submit on time-up
- Section nav sticky, passage panel scroll-locked
- Browser Web Speech API untuk audio listening (TTS, tidak ada audio file
  licensing issue)
- Confirm modal sebelum submit
- Anti-cheat: server strip `correctAnswer` dari questionsJson sebelum kirim

### 9.5 Hasil

`/karir/ielts/result/[id]` — band score circle + label (Modest/Good/Expert
User), breakdown per section, expandable answer review (jawaban kamu vs
benar untuk yang salah).

Disclaimer: estimasi ±0.5-1.0 band vs official, bukan resmi.

## 16. Catatan Penting

### LPDP 2026 — perubahan format

Format Reguler/Targeted/Afirmasi diganti menjadi **STEM Industri Strategis**
+ **SHARE (Social, Humanities, Art, Religious Studies, Economics)**. Dana
abadi naik Rp25 triliun jadi total Rp180 triliun. Pelamar yang familiar
format lama harus baca ulang requirement.

### PALM Australia — Indonesia TIDAK eligible

⚠ PALM Scheme hanya untuk 9 negara Pasifik + Timor-Leste. Entry PALM di DB
sudah disertai warning prominent + alternatif (WHV 462, IA-CEPA Skills
Exchange).

### IELTS — disclaimer akurasi

Band score adalah **estimasi ±0.5-1.0** dibanding official British Council
/ IDP / IELTS USA. Audio listening pakai browser TTS, bukan rekaman
profesional. Cocok untuk persiapan, bukan tes resmi.

### Periodic scraping — biaya

Cost per run estimasi (untuk 1 source dengan ~20 URL discovered):
- Brave Search: ~$0.005 (gratis tier 2000/bulan)
